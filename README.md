# Job Openings App

A simple React job listing application that displays multiple company job openings using reusable card components.

## Features

- Reusable `Card` component
- Dynamic rendering using `.map()`
- Displays company logo, role, pay, and location

## Tech Stack

- React
- JavaScript
- Lucide React


## Dependency

```bash
npm install lucide-react
```
